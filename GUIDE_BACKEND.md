## 12. Demarrage et Maintenance

### Commandes utiles

```bash
# Demarrer le backend
cd wibot-backend
docker-compose up -d

# Voir les logs
docker-compose logs -f

# Redemarrer un service
docker-compose restart n8n

# Arreter tout
docker-compose down

# Reinitialiser la base de donnees
docker-compose down -v
docker-compose up -d
```
